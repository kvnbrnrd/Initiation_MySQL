1) Afficher tous les gens dont le nom est palmer.
Solution : SELECT * FROM `bdd_1` WHERE `last_name` = 'Palmer';

2) Afficher toutes les femmes.
Solution : SELECT * FROM `bdd_1` WHERE `gender` = 'Female';

3) Tous les états dont la lettre commence par N.
Solution : SELECT * FROM `bdd_1` WHERE `state_code` LIKE 'N%';

4) Tous les emails qui contiennent google.
Solution : SELECT * FROM `bdd_1` WHERE `email` LIKE '%google.%';

5) Répartition par Etat et le nombre d’enregistrement par état (croissant).
Solution : SELECT state_code, count(*) FROM `bdd_1` GROUP BY state_code;

7) Nombre de femme et d’homme.
Solution : SELECT gender, count(*) FROM `bdd_1` GROUP BY gender;

8) Afficher l'âge de chaque personne, puis la moyenne d’âge générale, celle des femmes puis celle des hommes.
Solution : SELECT TIMESTAMPDIFF(year, STR_TO_DATE(birth_date, '%d/%m/%Y'), NOW()) AS age FROM `bdd_1`; SELECT AVG (TIMESTAMPDIFF(year, STR_TO_DATE(birth_date, '%d/%m/%Y'), NOW())) AS average FROM `bdd_1`; SELECT gender, AVG (TIMESTAMPDIFF(year, STR_TO_DATE(birth_date, '%d/%m/%Y'), NOW())) AS average FROM `bdd_1` GROUP BY gender
